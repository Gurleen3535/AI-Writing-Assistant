# ✨ AI Writing Assistant

An AI-powered writing assistant built using Python that helps users generate, summarize, and enhance text efficiently.

## 🚀 Features
- Text generation using AI (Groq API)
- PDF document reading (PyPDF2)
- Word file support (python-docx)
- User-friendly GUI (CustomTkinter)
- Summarization and content enhancement

## 🛠️ Technologies Used
- Python
- CustomTkinter
- Groq API
- PyPDF2
- python-docx

## 📂 Project Structure
- main Python file for application logic
- Supports multiple input formats (text, PDF, DOCX)
